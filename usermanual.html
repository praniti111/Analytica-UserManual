Analytica User Manual

    Given below are few Instructions which will help you understand and guide on how to use the ANALYTICA Website.

    What is ANALYTICA?


    Divisions

    Analytica has Registration amd Login for 2 profiles
      1. User
      2. Admin

    You can register as a User or an Admin according to your requirenment.

    User Registration/Login Functionality:

    A User can upload the sheet in the .csv, .xls, .xlsx format only.

    or

    The User can enter the data manually by using the Manual Entry button

    Admin Registration/Login Functionality:

    An Admin can Upload as well as see the analysis of the data which has been uploaded previously.

    There are Two sections

      1. Current Analysis: It contains the analysis of the currently uploaded data by the user or admin
      2. Total Analysis: It contains the analysis of the user as well as admin i.e Total analysis of the data

    An Admin can upload the sheet in the .csv, .xls, .xlsx format only.

    or

    The admin can enter the data manually by using the Manual Entry button

    General Instructions

    1. Sheet Format: The file should be in .csv, .xls, .xlsx format only.

    Manual Entry Instructions

      • All the Author’s Name should be in the format of First Name & Last Name.
      • Department Name should not be in abbreviation It should be mentioned in full form.
      Eg. : donot write CS ❌
      write Department of Computer Science. ✅

    Compulsory Fields to fill in Manual Entry

      1. Paid/Unpaid
      2. Department
      3. Author 1
      4. Public Score
      5. Faculty Score

      In Analysis, both in Total and Current, please note the details
      1. Faculty Analysis:
      Compulsory Fields to fill
      1. Author
      2. Year

      2. Department Analysis:
      Compulsory Fields to fill
      1. Author
      2. Year
